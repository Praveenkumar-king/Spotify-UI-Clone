# 🎵 Spotify UI Clone

> A modern, responsive Spotify-inspired music streaming interface built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project recreates the visual experience of a music streaming platform, including the home page, sidebar navigation, playlists, album views, search interface, and music player UI.

> **Disclaimer:** This project is an independent educational UI clone created solely for learning and portfolio purposes. It is **not affiliated with, endorsed by, or associated with Spotify**.

---

# 📖 Project Overview

The Spotify UI Clone recreates the modern desktop and mobile interface of a music streaming platform while demonstrating responsive web development, reusable UI components, CSS layouts, and JavaScript interactions.

This project focuses on creating an active user experience with elegant typography, card-based layouts, interactive navigation, and a fully interactive music player interface.

---

# ✨ Features

## Home
- Featured Playlists
- Recently Played
- Made For You
- Trending Albums
- Popular Artists
- New Releases

## Sidebar Navigation
- Home
- Search
- Your Library
- Interactive Playlists List

## Search
- Search Bar
- Browse Categories
- Live search input filter matching title/artist

## Music Player (Fully Functional Mock)
- Album Artwork & Track Info
- Dynamic Play/Pause, Next, Previous track actions
- Seek bar timeline progress tracking
- Volume levels control (mute/unmute, slider)
- Shuffle and Repeat states toggling
- Synchronization with HTML Audio element playing actual music!

## Playlist & Album Page
- Playlist Banner with custom colored backgrounds
- Followers statistics
- Dynamic tracks table list
- Add songs to favorites

## Artist Page
- Artist banner with monthly listeners
- Pop tracks ranking list
- Related albums grids
